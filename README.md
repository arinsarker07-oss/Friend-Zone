# 🚀 Friends Zone - Social Networking Platform

Friends Zone is a modern, high-performance social connectivity dashboard built with **Next.js 15** and **Tailwind CSS**. It allows users to track their contacts, monitor activities through a timeline, and view analytics in a sleek, user-friendly interface.

**Live Demo:** [https://friends-zone-theta.vercel.app/](https://friends-zone-theta.vercel.app/)

---

## ✨ Features

-   **Interactive Dashboard:** View overall statistics and contact insights at a glance.
-   **Activity Timeline:** A real-time activity tracker with dynamic date/time synchronization.
-   **Contact Management:** Categorized contact views (All, Athlete, Singer, etc.) with professional card layouts.
-   **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop using Tailwind CSS.
-   **Modern Tech Stack:** Leveraging Next.js Server Components and Client Components for optimal performance.
-   **Professional UI:** Integrated with DaisyUI and Lucide/React-Icons for a polished look.

---

## 🛠️ Tech Stack

-   **Framework:** [Next.js 15 (App Router)](https://nextjs.org/)
-   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
-   **UI Components:** [DaisyUI](https://daisyui.com/)
-   **Icons:**  [React Icons](https://react-icons.github.io/react-icons/)
-   **Deployment:** [Vercel](https://vercel.com/)
-   **State Management:** React Context API
